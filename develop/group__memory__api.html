Detailed Description

Flash and EEPROM access and management functionality.

This module provides functions for accessing and managing non-volatile memory, including:

  • Flash memory read/write operations
  • EEPROM data storage
  • Device state and configuration storage
  • Sensor readings storage
  • Logging data management
