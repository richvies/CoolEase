HID configuration init callback.

Called by libopencm3 - usb_standard_set_configuration() to initialize HID configuration

  • Initializes endpoints, IN (0x01) & OUT (0x81), and sets the transaction callbacks for both endpoints. hid_report_callback()
  • Sets control request callback hid_control_request() which is called by libopencm3 usb_standard_set_configuration() to configure as HID device
